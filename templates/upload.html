Upload Excel Data

Upload your Excel file containing property data. The system will automatically process the file and import the data into the database.

Please select an Excel file to upload.
Supported formats: .xlsx, .xls (max 50MB)
Data Format

The file should contain property data in the following format:

  • Property Address: Usually located in the first few rows of each sheet
  • Date: Transaction or work dates
  • Description: Description of work or transaction
  • Amount: Cost or income amount
  • Method: Payment method (optional)

Multiple properties can be in separate sheets within the same Excel file.
